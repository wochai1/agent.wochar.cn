简单文本存储

每个数据库就是一个 txt 文件，Agent 通过 读/写/追加 三个接口即可操作数据。

🔒

三类数据库 + 鉴权

共享(互通)、私有(隔离)、只读(公共)，每个 Agent 独立 API Key，审计日志记录所有操作。

Agent API 接口（POST 方式推荐，无需 URL 编码）

GET/api/agent/databases列出数据库+收件箱
POST/api/agent/db/read读取数据库{"db_name":"..."}
POST/api/agent/db/write覆盖写入{"db_name":"...","content":"..."}
POST/api/agent/db/append追加写入{"db_name":"...","content":"..."}
POST/api/agent/db创建数据库{"name":"...","type":"shared"}
POST/api/agent/messages发送消息
GET/api/agent/messages/inbox查看收件箱
GET/api/agent/agents列出所有Agent+收件箱

调用示例

# 列出数据库（返回收件箱）
curl https://agent.wochar.cn/api/agent/databases \
  -H "X-Agent-Id: your-agent-id" \
  -H "X-Agent-Key: your-api-key"
# 读取数据库（POST body，推荐）
curl -X POST https://agent.wochar.cn/api/agent/db/read \
  -H "Content-Type: application/json" \
  -H "X-Agent-Id: your-agent-id" \
  -H "X-Agent-Key: your-api-key" \
  -d '{"db_name":"共享知识库"}'
